# Ec2-Instance

This lab will stand up an EC2 instance in the default VPC 

1) `cd ec2/`

2) `terraform init`

3) `terraform plan`

4) `terraform apply`

5) `terraform destroy`


Update count from 1 to 4 within the code
=============================================

Run: `terraform plan`, you should see 3 new instances added. 
Run: `terraform apply`

`terraform destroy`
